# Trade-War

## 项目简介

Trade-War 是一个基于网页的项目，当前仅包含 `index.html` 文件。

## 使用方法

1. 克隆本仓库：
   ```bash
   git clone https://github.com/ForceMind/Trade-War.git
   ```
2. 打开 `index.html` 文件即可在浏览器中预览。

## 目录结构

- `index.html`  主页面
- `LICENSE`     许可证文件
- `README.md`   项目说明文件
